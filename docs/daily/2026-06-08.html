🛠️ Glaze — 一句话生成 Mac 应用并一键上架
一句话

Raycast 团队出品，输入需求自动生成 macOS 原生应用，内置开发、日志、上架全链路。

链接

https://glaze.app ↗

怎么玩

前往 glaze.app 申请内测资格，收到邀请后安装 Glaze ——它是 Raycast 团队的新作，集成了 AI 编码、调试日志查看和 App Store 发布功能。打开后直接用自然语言描述你想要的应用（比如"帮我做一个用 Spotify API 的音乐电台"），AI 会自动生成完整项目代码、构建并打包。你可以在内置的 Store 里一键安装其他用户分享的 App，体验接近原生 macOS 应用。开发者用 10 分钟、一句话需求就做出了可用的 Spotify 电台 App，debug 只做了一次。

为什么值得关注

Raycast 已是 Mac 效率工具标杆，Glaze 把 vibe coding 推进到了「vibe shipping」——不再仅仅生成代码片段，而是覆盖了从需求到发布的完整链条。上下文全打包的设计意味着 AI 能看到项目全局，开发效率远超在 IDE 里手动复制粘贴代码。

应用场景
  • 快速原型验证：产品想法 → 10 分钟可交互原型

🛠️ Cursor Design（Claude Design 本地运行版）
一句话

Cursor 新增浏览器预览 + 元素标注功能，描述一个页面 → 生成 HTML → 点击任意元素说出修改需求。

链接

https://cursor.com ↗

怎么玩

更新 Cursor 到最新版，在终端执行 `npx skills add JimLiu/baoyu-design` 安装宝玉开发的 baoyu-design Skill。打开 Cursor，使用 Composer 2.5 模式，输入设计需求（如"帮我设计一个 Reader Mac App 的信息架构"），它会先生成 3 个设计变体供你选择。选择后进入浏览器预览，点击任意 UI 元素就能通过自然语言调整样式——不需要写 CSS。搭配 Claude Opus 4.8 效果最佳，宝玉实测对比发现 Opus 审美显著优于 GPT-5.5。Knobs 模式下点击画布元素可直接拖拽调整 CSS 参数。

为什么值得关注

这是 Claude Design 的本地化替代方案，不依赖浏览器扩展，直接在 IDE 内完成设计→开发闭环。元素标注（element annotation）+ Knobs 调节让非设计师也能产出专业级 UI，且所有代码都在本地可修改。

应用场景
  • 独立开发者快速出高颜值 Landing Page
  • 前后端分离项目的前端快速搭建

🛠️ Repo2RLEnv — 把任意 GitHub 仓库变成 RL 训练环境
一句话

HuggingFace 开源工具，从 GitHub 仓库自动合成可验证的强化学习训练数据。

链接

https://huggingface.co ↗

怎么玩

安装该工具后，只需指向任意 GitHub 仓库地址，它会自动分析每个已合并的 PR——提取 PR 描述作为「任务描述」、把合并前的代码作为「初始状态」、合并后的 diff 作为「正确解法」，生成可验证的 RL 训练任务。生成的数据集会自动推送到 HuggingFace Hub，可直接用于训练代码生成 Agent。核心理念是将真实的软件工程实践转化为 RL 信号：每个成功的 PR 都是一个「从问题状态到达目标状态」的轨迹。

为什么值得关注

代码 Agent 训练长期受困于高质量 RL 数据的稀缺，Repo2RLEnv 把 GitHub 上亿级真实 PR 变成了训练信号源。不是合成数据，而是真实开发者在真实项目中解决问题的路径——这比任何人工构造的编程题都更有价值。HuggingFace 出品意味着与现有训练栈（TRL、Axolotl）无缝集成。

应用场景
  • 为自己的开源项目生成代码修复训练数据
  • 训练专用编程 Agent（如某框架/语言专项能力）
