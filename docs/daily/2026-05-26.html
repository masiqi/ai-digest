🛠️ Grok Build CLI（xAI）
一句话

xAI 出品的 AI 编程 Agent 命令行工具，对标 Claude Code 和 OpenAI Codex，SuperGrok / X Premium+ 用户免费使用订阅额度。

链接

https://x.ai/cli ↗

怎么玩

执行 curl -fsSL https://x.ai/cli/install.sh | bash 一键安装，然后运行 grok 启动交互式 TUI（终端全屏、鼠标操作）；也支持 Headless 模式 grok -p "指令" 放入脚本自动跑。支持切换任意自定义模型，不绑死 Grok。同款模型 grok-build-0.1 已在 API 上开放。macOS / Linux / Windows 都支持。核心模式包括 Plan（任务规划）和 Build（代码生成执行）。

为什么值得关注

Agent CLI 赛道 Claude Code 和 Codex 二分天下已久，xAI 凭借 Grok 模型的大尺度能力和 X 平台用户基础直接入局，且安装门槛极低（一条 curl 命令），对已有 X Premium 订阅的用户零边际成本。内置 TUI 交互和 Headless 脚本双模式，兼顾手动和自动化场景。

应用场景
  • 日常代码开发——在终端里让 Grok Build 分析项目、生成代码、运行测试
  • CI/CD 流水线——Headless 模式下将 grok -p 嵌入自动化脚本做代码审查或修复
